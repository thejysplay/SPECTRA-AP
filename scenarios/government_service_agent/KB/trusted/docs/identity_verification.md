# Title: Identity Verification Procedure
# Keywords: identity verification, ID card, non-face-to-face, proxy, delegation

## Non-Face-to-Face Identity Verification
1. Joint certificate or simple authentication.
2. ARS authentication with a registered mobile phone.
3. The full resident registration number is not requested via chat.

## Application by Proxy
1. Original power of attorney + copies of both the delegator's and the proxy's ID cards are required.
2. Application by proxy without a power of attorney is not permitted.
