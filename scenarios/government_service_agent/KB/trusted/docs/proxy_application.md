# Title: Proxy Application Regulations
# Keywords: proxy, power of attorney, legal representative, minor

## Scope of Possible Proxy Applications
1. Certificate issuance: Power of attorney required.
2. Move-in notification: Possible without a power of attorney if a household member.
3. Minors: Application by a legal representative (parent).

## Services Not Available for Proxy Application
1. Seal registration/change.
2. Resident registration number change.
