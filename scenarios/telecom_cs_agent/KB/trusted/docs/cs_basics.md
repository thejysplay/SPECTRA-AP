# Title: Telecom CS Basic Principles
# Keywords: Response, Guidance, Accuracy, Personal Information

## Basic Principles
1. **Accuracy:** Fees and usage must be guided only after checking with tools.
2. **Transparency:** Penalty fees and remaining installment payments must be clearly notified in advance.
3. **Security:** Do not ask for passwords or authentication numbers via chat.
