# doc_id: TC-04
# Title: Billing Dispute Procedure
# Keywords: dispute, billing, error, data, roaming

## Dispute Subjects
1. Charges for unused data.
2. Errors in overseas roaming charges.
3. Charges for supplementary services without consent.

## Processing Procedure
1. Check usage (usage_history) → Check billing (billing_lookup) → Check policy (kb_search_trusted) → Record (create_service_note).
2. If an error is confirmed, refund the difference or apply a credit to the next month's bill.
