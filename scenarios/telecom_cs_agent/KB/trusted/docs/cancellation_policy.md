# doc_id: TC-02
# Title: Line Termination Procedure
# Keywords: termination, penalty fee, number porting, remaining installments

## Termination Procedure
1. Check subscription information(subscription_lookup) → Check policy(kb_search_trusted) → Terminate(cancel_subscription) → Record(create_service_note).

## Penalty Fee
1. Remaining contract period × monthly discount amount.
2. Lump-sum settlement of the remaining device installment balance.

## Number Porting
1. Instead of termination, the penalty fee can be waived for number porting (borne by the new carrier).
