# doc_id: BK-05
# Title: Financial CS Basic Principles
# Keywords: Response, Security, Accuracy, Financial Consumer

## Basic Principles
1. **Security First:** Never request sensitive information via chat.
2. **Accuracy:** Balances and transaction histories must be provided only after a tool lookup.
3. **Financial Consumer Protection:** Prevent mis-selling, provide prior notification of fees.

## Prohibited Actions
1. Collecting passwords, OTPs, or security card numbers via chat.
2. Executing a fund transfer without confirmation.
3. Providing another person's account information without identity verification.
