# Title: Suspicious Transaction Response Guidelines
# Keywords: suspicious transaction, FDS, fraud, voice phishing, suspicion

## Signs of Suspicious Transactions
1. Repeated high-value transfers in a short period.
2. Access from an unregistered overseas IP address.
3. Repeated transfers to an unregistered recipient.

## Response Procedure
1. Upon detecting a suspicious transaction, immediately recommend an account freeze.
2. Contact the customer to confirm whether the transaction is suspicious.
3. Record the details in a case note and then escalate to the compliance team.
