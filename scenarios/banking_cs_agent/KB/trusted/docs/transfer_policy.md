# Title: Transfer Limits and Regulations
# Keywords: transfer, limit, daily limit, single transaction limit, overseas remittance

## Transfer Limits
1. **Single Transaction Limit:** Default KRW 5,000,000. Identity verification required to increase the limit.
2. **Daily Limit:** Default KRW 10,000,000. Branch visit required to exceed this limit.
3. **Overseas Remittance:** USD 5,000 or less per transaction. Foreign exchange transaction report required for amounts exceeding this.

## Transfer Execution Conditions
1. Transfer not possible if balance is insufficient.
2. Transfer proceeds after verifying recipient account validity.
3. Record in case note after transfer completion.
