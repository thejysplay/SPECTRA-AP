# doc_id: BK-03
# Title: Identity Verification Regulations
# Keywords: identity verification, authentication, remote, ID card

## Remote Identity Verification
1. ARS authentication using the registered mobile phone number.
2. Public key certificate or biometric authentication.
3. Do not request passwords, OTPs, or security card numbers via chat/phone.

## Tasks Requiring Identity Verification
1. Changing transfer limits.
2. Registering/changing beneficiaries.
3. Unfreezing an account.
