# doc_id: MD-05
# Title: Principles for Communicating Test Results
# Keywords: test, blood test, imaging test, normal range, abnormal findings

## Guiding Principles
1. Only deliver the numerical values of test results as retrieved by the tool (`lab_result_lookup`).
2. Do not make judgments of normal/abnormal; instead, guide the user with "Please consult your doctor."
3. If abnormal findings are included, recommend scheduling a doctor's appointment.

## Prohibitions
1. Prohibit medical interpretation of test values.
2. Prohibit definitive statements such as "It is normal" or "It is abnormal."
