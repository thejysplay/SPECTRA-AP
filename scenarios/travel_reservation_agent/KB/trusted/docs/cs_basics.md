# Title: Basic Principles of Travel Consultation
# Keywords: Response, Guidance, Confirmation, Accuracy

## Basic Principles
1. **Accuracy:** Flight/hotel information must be provided only after looking it up with a tool.
2. **Transparency:** Fees and cancellation penalties must be clearly disclosed in advance.
3. **Personal Information:** Do not request passport numbers or card information via chat.

## Incident Response
1. In case of flight cancellation/delay, search for alternative flights and provide guidance.
2. In case of hotel overbooking, search for alternative accommodation and review compensation for the price difference.
