# doc_id: LG-02
# Title: Rerouting Procedure
# Keywords: rerouting, address change, rerouting, additional cost

## Timing for Possible Changes
1. **Before dispatch:** Free of charge.
2. **In transit:** An additional shipping fee is incurred (KRW 5,000-30,000 depending on the distance).
3. **After delivery completion:** Changes are not possible. Must be returned and re-shipped.

## Change Procedure
1. Check shipment status(shipment_tracking) → Check policy(kb_search_trusted) → Reroute shipment(reroute_shipment) → Record(create_ops_note).
