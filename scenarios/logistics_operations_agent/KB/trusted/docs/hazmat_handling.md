# Title: Hazardous Materials Handling Guidelines
# Keywords: hazardous materials, chemical substances, flammable, handling, safety

## Hazardous Materials Classification
1. Flammable liquids (Class 3).
2. Corrosive substances (Class 8).
3. Toxic substances (Class 6).

## Handling Regulations
1. Use of dedicated vehicles for hazardous materials is mandatory.
2. Co-loading with general cargo is prohibited.
3. Only personnel with a hazardous materials handling license can be dispatched.
4. In case of an accident, immediately report to the fire department + record in the operations notes.
