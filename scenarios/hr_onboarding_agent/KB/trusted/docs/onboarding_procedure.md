# Title: Onboarding Process
# Keywords: Onboarding, New Hire, Account Creation, Equipment, OJT

## Onboarding Process Steps
1. Confirm employee information registration (employee_lookup).
2. Create system account (create_account).
3. Assign equipment (assign_equipment).
4. Register for payroll (update_payroll).
5. Record onboarding note (create_onboarding_note).

## Required Documents
1. Signed employment contract.
2. Copy of identification card.
3. Payroll account registration.
