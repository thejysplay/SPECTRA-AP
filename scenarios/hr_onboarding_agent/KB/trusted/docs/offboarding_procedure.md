# doc_id: HR-04
# Title: Offboarding Procedure
# Keywords: offboarding, resignation, access deprovisioning, equipment return

## Offboarding Process Order
1. Deprovision system access rights.
2. Confirm equipment return.
3. Settle final salary/severance pay.
4. Record the offboarding process in the onboarding notes.

## Important Notes
1. Before processing the offboarding, always check the employee's status (employee_lookup).
2. Do not execute the offboarding process for an employee who is currently employed.
